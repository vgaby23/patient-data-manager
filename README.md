# Patient Data Manager
This program performs CRUD action and also provide insights on hospital patients data.

Things to do before running the program:
1. Adjust .env values for database credential
2. Activate virtual environment before running

The program allows user to perform CRUD action on several data:
1. Patient Data
2. Appointments
3. Treatments Schedule
4. Bills

User can also see the performance and demographics of the hospital and patients through analytics menu.
